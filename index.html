Forth Engine

Forth Engine is an open-source C++ Engine to render and manipulate high, four dimensional objects into three dimensional objects that can be rendered via ordinary graphics renderer, like OpenGL.

Forth Engine is a successor to Engine4, and currently it's in main development phase.
